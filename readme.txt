A lightweight and fast URL shortener plugin for WordPress using the is.gd API.

== Description ==

Nepsus Link Shortener allows you to quickly convert long URLs into short, shareable links using a simple shortcode.

This plugin is designed to be:
- Lightweight and fast
- Secure (nonce protected)
- No database usage
- Clean and modern UI
- Beginner-friendly

= Shortcode Usage =
Use the following shortcode anywhere:
[nepsus_url_shortener]

== Installation ==

1. Upload the `link-shortener` folder to `/wp-content/plugins/`
2. Activate the plugin through the Plugins menu
3. Add `[nepsus_url_shortener]` to any post or page

== Frequently Asked Questions ==

= Does this plugin store URLs in the database? =
No. URLs are generated in real time using the is.gd API.

= Is this plugin free to use? =
Yes, it is completely free.

= Does it require an API key? =
No API key is required.
